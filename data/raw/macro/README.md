## Macro & Market Data

**Source:** FRED, Yahoo Finance  
**Period:** 2021-01-01 ~ 2025-12-31

### Files

| File | Description |
|------|-------------|
| `macro_data.csv` | Monetary policy & inflation indicators |
| `additional_data.csv` | Market volatility & benchmark indices |

### Fields — macro_data.csv

| Column | Description |
|--------|-------------|
| Date | Date |
| Federal_Funds_Rate | US federal funds rate |
| Fed_Balance_Sheet | Federal Reserve balance sheet size |
| CPI | Consumer Price Index |
| Treasury_10Y_Yield | 10-year US Treasury yield |

### Fields — additional_data.csv

| Column | Description |
|--------|-------------|
| Date | Date |
| SP500_ETF | S&P 500 ETF price (benchmark) |
| WTI_Oil | WTI crude oil price |
| VIX | S&P 500 volatility index |
| VXN | NASDAQ volatility index |
| VVIX | Volatility of VIX |
